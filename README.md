## ✨ Features

- User Authentication
- Product Catalog
- Product Search
- Shopping Cart
- Stripe Payments
- Cash on Delivery
- Order Tracking
- Admin Dashboard
- Product Management
- Order Management
- Image Upload with Cloudinary
- Responsive Design

## 🚀 Installation

```bash
git clone https://github.com/yourusername/theNorthFace.git

cd theNorthFace
```

Install dependencies

```bash
cd frontend
npm install

cd ../backend
npm install

cd ../admin
npm install
```

Run the project

```bash
# Backend
npm run server

# Frontend
npm run dev

# Admin
npm run dev
```
